# python-calculator
A Python Calculator with logging to a CSV file, supporting +, -, *, /, **, and %, with error handling for Invalid Operators, Value Errors, and Division by Zero.

## Features
- Addition, Subtraction, Multiplication, Division, Exponentiation, Modulus
- Logs each calculation to a CSV file
- Gracefully Deals with multiple errors

## How to run
- Install Python:
    - Open Command Prompt
    - winget install Python.Python.3
    - Check if Python is installed with 'python --version'
- Clone the project:
    - Copy and run in Command Prompt:
    - git clone https://github.com/yashr0507/python-calculator.git
- Open the folder python-calculator
- Enter 'python python-calculator.py' in Command Prompt
- Follow the Menu and enjoy!
